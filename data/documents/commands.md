# Пользовательские команды

JARVIS понимает эти фразы и сценарии.

## Личные вопросы (ответ из базы знаний)

| Фраза | Ожидаемое поведение |
|-------|---------------------|
| «кто я» / «кто я бро» / «братан» | Имя, вуз, город из about_me.md |
| «где я учусь» | СибГУ, ИиВТ, 2 курс |
| «что ты знаешь обо мне» | Краткая сводка из документов |
| «какие у меня проекты» | JARVIS и учебные проекты |

## Системные

| Фраза | Действие |
|-------|----------|
| «статус системы» | Ollama + размер индекса + время |
| «что ты умеешь» | RAG, tools, чат |
| «переиндексируй» | Напомнить: `jarvis index` |

## Учёба

| Фраза | Действие |
|-------|----------|
| «какие дедлайны» | Из study.md |
| «помоги с SQL» | Подсказки по SQL из study.md и skills.md |

## Сленг

- **бро**, **братан** — дружеское обращение, не вопрос про брата
- **база** — база знаний / документы в data/documents
